優質密碼設定準則

建立新密碼時請遵循下列方針：
  • 密碼長度最少6碼，最長12碼
  • 需英文字、數字夾雜，不能為全是英文字或數字
  • 使用英文字可全為大寫或全為小寫，若使用大、小寫夾雜時，輸入時必須注意大、小寫區分
  • 不可使用特殊符號，例如：!@#$%^;:?
  • 不可為您身份證字號
  • 不可為相同英文字、數字連續達4次(含)以上，例如：1111、AAAA
  • 不可為連續性之英文字、數字達4次(含)以上，例如：1234 或 4321 或 abcd 或 dcba
  • 不可重複使用前3次舊密碼
  • 每90天必須變更一次密碼
  • 連續輸入密碼錯誤達5次，帳號將鎖住。
  1. 使用 8 至 16 個字元，至少必須符合下列 3 項條件：
    • 小寫字母
    • 大寫字母
    • 數字 (0-9)
    • 符號，包含：! @ # $ % ^ & * - _ + = [ ] { } | \ : ‘ , . ? / ` ~ “ < > ( ) ;
  2. 不可重覆使用前一次舊密碼。
  3. 每90天必須變更一次密碼。
